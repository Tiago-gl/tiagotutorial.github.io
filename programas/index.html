JavaScript® (às vezes abreviado para JS) é uma linguagem leve, interpretada e baseada em objetos com funções de primeira classe, mais conhecida como:

  • a linguagem de script para páginas Web
  • O JavaScript é uma linguagem baseada em protótipos
  • suportando estilos de orientação a objetos, imperativos e declarativos
